# Pandas Sales Analysis

## 概要
本プロジェクトでは、売上データと顧客マスタを結合し、顧客別の売上集計結果をCSVとして出力する処理を実装しています。
顧客の売上規模を可視化し、業務上の意思決定に活用できるデータを作成しています。
## 使用技術
- pandas
- Python 
- CSV
## データ構成
- sales_data.csv:売上明細データ
- customers.csv:顧客マスタ
- final_result.csv:最終出力データ
## 処理内容
- groupby + agg による顧客単位の売上集計
- merge(left) による顧客情報の付与
- sort_values による売上順ソート
- 欠損データの判定、除外処理
- 売上規模に応じた顧客ランク
- 集計結果をCSVとして出力

## 実行方法
```bash
python main.py
```

## 工夫した点

### left join を採用した理由
売上データを基準とし、売上が発生したデータを失わないために left join を採用しています。  
これにより顧客マスタ側に欠損がある場合でも、売上データ自体は保持される設計としました。

### 欠損データへの耐性
merge 後に欠損値が発生することを前提とし、欠損判定、除外処理を行っています。  
実務を想定し、処理途中でエラーにより停止しないことを重視しました。
